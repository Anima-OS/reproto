Index — Package: io.reproto.toystore-1.0.0

type Toy

A toy in a toy store.

Examples

{
  "id": 4122,
  "name": "LEGO Pirates of the Caribbean",
  "category": {
    "id": 10010,
    "name": "Lego"
  },
  "tags": ["plastic", "expensive"],
  "status": "available"
}

field id : u64

Identifier of the toy.

field name : string

Name of the toy.

field category : Category

Category of the toy.

field tags : [ Tag ]

Tags of the toy.

field status : Status

The status of the toy in the store.
